HEADINGS

In HTML, there are six different headings, heading elements. Headings can be used for a variety of purposed, like titling sections, articles or other forms of content.

The following is the list of heading elements available in HTML. They are ordered from largest to smallest in size.

  1. <h1> - used for main headings. All other smaller headings are used for subheadings.
  2. <h2>
  3. <h3>
  4. <h4>
  5. <h5>
  6. <h6>


The following example code uses a headline intended to capture a reader's attention. It uses the largest heading available, the main heading element.

<h1>BREAKING NEWS</h1>

  2. Divs

    One of the most popular elements in HTML is the <h1>. <h1> is short for "division" or container that divides the page into sections.These sections are very useful for grouping elements in your HTML together.
  3. Attributes

    Attributes are content added to the opening tag of an element and can be used in several different ways, from providing information to changing styling. Attributes are made up of the following two parts
    • The name of the attribute
    • The value of the attribute

    List of Attributes
    1. id

      2. One commonly used attribute is the id. We can use the id attribute to specify different content (such as <div>S) and is really helpful when you use an element more than once.

      When we add an id to a <div>, we place it in the opening tag:

      <div id="intro">
      <h1>Introduction</h1>
      </div>

    2. class - <div class="title></div>
    3. Image Alts

      The alt attribute, which means alternative text, brings meaning to the images on our sites. The alt attribute can be added to the image tag just like the src attribute. The value of alt should be a description of the image


      <img src="#"A field of yellow sunflowers"/>
  4. VideoS
